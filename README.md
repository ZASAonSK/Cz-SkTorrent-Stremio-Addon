SKTorrent + TorBox (Multi‑User) Stremio Addon

Node.js/Express Stremio addon pre SKTorrent, ktorý podporuje multi-user konfiguráciu cez URL (Base64), vyhľadávanie torrentov, filtrovanie pre seriály (SxxExx / xxyy) a voliteľnú integráciu TorBox (cache check + proxy routy pre prehrávanie/sťahovanie).

Funkcie

    Multi-user config bez pevne natvrdo uložených cookies (UID/pass sa posielajú v konfigurácii URL).

    Vyhľadávanie na SKTorrent + parsovanie výsledkov (cheerio).

    Výber streamov podľa seedov, veľkosti a kategórie.

    Seriály: filtrovanie podľa série/epizódy a výber správneho súboru z .torrent (fileIdx).

    TorBox (voliteľné): kontrola cached torrentov a presmerovanie na priamy link cez proxy endpointy.
